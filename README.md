# الشورة (Shurah)

منصة لاتخاذ القرار بالاعتماد على عدة نماذج ذكاء اصطناعي.

> لا نجمع الإجابات. نحن نبني قرارًا.

## الرؤية

أن تصبح "الشورة" الطبقة الذكية لاتخاذ القرار فوق جميع نماذج الذكاء الاصطناعي.

## البنية

```
shurah/
├── apps/           # تطبيقات (API, CLI)
├── packages/       # حزم
│   ├── decision_engine/  # محرك اتخاذ القرار
│   ├── shared/           # مشترك
│   └── adapters/         # محولات الخدمات الخارجية
├── tests/          # اختبارات
└── docs/           # وثائق
```

## التطوير

```bash
pip install -e ".[dev]"
pytest
ruff check
mypy .
```
