# ADR-001: Project Foundation

## الحالة
مقبول

## السياق
تأسيس مشروع "الشورة" كمنصة لاتخاذ القرار باستخدام نماذج الذكاء الاصطناعي.

## القرار
1. استخدام Python 3.13 مع strict typing
2. استخدام Pydantic v2 للـ Contracts
3. استخدام Ruff + mypy strict للجودة
4. استخدام pytest للاختبارات
5. هيكل Monorepo مع `packages/` و `apps/`
6. اعتماد Conventional Commits

## العواقب
- إيجابية: نوعية كود عالية، قابلية اختبار، فصل واضح للمسؤوليات
- سلبية: وقت إعداد أطول في البداية
